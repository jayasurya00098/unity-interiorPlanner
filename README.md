# Interior Planner and Viewer (WebGL/AR)

This is a simple interior planner project. There are two parts for this project,
 - Planner (WebGL)
 - Viewer (Mobile AR)


## Tech Stack

**Unity:** 2023.1.0b2

**Third-Party:** Lean Touch

**Server:** [Crud Crud](https://crudcrud.com/)
